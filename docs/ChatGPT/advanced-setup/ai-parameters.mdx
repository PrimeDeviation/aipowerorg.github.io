# AI Parameters

You can change the AI engine parameters of the chatbot.

The following table provides an overview of the parameters that are used by the chatbot.

| Parameter           | Description                                                   | Default Value | Range |
|---------------------|---------------------------------------------------------------|---------------|-------|
| [Model](/docs/ai-engine/openai/gpt-models)               | The AI engine used by the chatbot.                            | Davinci       | N/A   |
| [Temperature](/docs/ai-engine/openai/temperature)        | Controls the randomness. Higher values mean more randomness.  | 0.7           | 0-1   |
| [Max Tokens](/docs/ai-engine/openai/max-tokens)          | Maximum number of tokens to generate.                         | 700           | 1-1400|
| [Top P](/docs/ai-engine/openai/top-p)               | Controls the diversity. Lower values mean more predictability.| 1.0           | 0-1   |
| [Best Of](/docs/ai-engine/openai/best-of)             | Number of different completions to return.                    | 1             | 1-20  |
| [Frequency Penalty](/docs/ai-engine/openai/frequency-penalty)   | Controls the frequency. Lower values mean more frequency.     | 0.01          | 0-2   |
| [Presence Penalty](/docs/ai-engine/openai/presence-penalty)    | Controls the presence. Lower values mean more presence.       | 0.01          | 0-2   |

You can learn more about each of these parameters in the [AI Engine](/docs/category/openai) section.

:::tip

Keep your **max token** value between **500-700** to avoid errors like:

*"This model's maximum context length is 4097 tokens. However, you requested 4134 tokens (2634 in the messages, 1500 in the completion). Please reduce the length of the messages or the completion."*

:::
