# Styling the Chatbot

You can easily customize the appearance of your chatbot by adjusting its font size, font color, background color, button color, text field background and border colors, user and AI message background colors, width and height, and even the avatars.

- **Font Size**: You can set the font size of the chatbot. Default is 13. It can be set to a value between 10 and 30.
- **Font Color**: You can set the font color of the chatbot. Default is #000000.
- **Background Color**: You can set the background color of the chatbot.
- **Button Color**: You can set the color of the buttons.
- **Text Field Background**: You can set the background color of the text field. Default is #ffffff.
- **Text Field Border**: You can set the border color of the text field. Default is #ffffff.
- **User Background Color**: You can set the background color of the user messages. Default is #444654.
- **AI Background Color**: You can set the background color of the user messages. Default is #343541.
- **Width**: You can set the width of the chatbot. Default is 300. It can be set to a value between 200 and 800. You can also specify the width in percentage.
- **Height**: You can set the height of the chatbot. Default is 400. It can be set to a value between 200 and 800. You can also specify the height in percentage.
- **Use Avatars**: You can enable or disable the avatars. If you enable this option, the chatbot will display the avatars of the user and the chatbot. If user is logged in then it will display the avatar of the user.
- **AI Avatar**: You can set the avatar of the chatbot. You can use the default one or you can upload your own avatar. Size must be 40x40.
- **Border Radius - Window**: This attribute allows you to adjust the curvature of the chatbot window corners. The default value is set to 20px.
- **Text Field Height**: This determines the height of the message text input field. By default, it's set to 60px.
- **Border Radius - Text Field**: You can alter the curvature of the corners of the message text input field. The default value is 20px.
- **Icons Color**: This attribute allows you to modify the color of the icons in the download bar. Default color is blue.
- **AI Thinking Text Color**: This allows you to set the color of the text displayed when the AI is processing a response.

All the style settings mentioned above apply both to the shortcode and the widget.

However, the widget has three additional options:

- **Icon**: This is your widget's icon, which is located on the right or left side of your website. The default size is 75x75. You can either use the default icon or upload a custom one.
- **Position**: This allows you to set the position of the widget on your website. You have two options: **left** or **right**.
- **Delay Time**: This is the time in seconds before your widget automatically appears on the screen. For example, if you set the delay time to 5, the widget will appear after 5 seconds. If you don't want to use this feature, simply leave the field empty.

Here are the steps to adjust the style of your chatbot:
